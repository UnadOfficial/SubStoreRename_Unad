# RenameByUnad

# 自用 请勿用于生产环境 Readme由Qwen生成
# SubStore中使用直接复制到单条订阅-脚本操作就行

一个用于重命名代理节点的 JavaScript 脚本，适用于 Surge、Clash 等代理配置工具。

## 功能特点

- **自动识别国家/地区**：根据节点名称中的关键词（如"美国"、"🇺🇸"、"Los Angeles"等）自动识别并标记国家/地区代码
- **标准化命名格式**：将节点名称统一格式化为 `订阅名 国家代码 - 编号 - 倍数` 的形式
- **去重处理**：自动检测并重命名重复的节点名称，添加序号后缀
- **支持多语言**：支持中文、英文、国旗表情等多种形式的国家/地区标识

## 使用方法

### 在 Surge 中使用

1. 打开 Surge 配置文件
2. 在 `[Script]` 部分添加：

```ini
[Script]
RenameByUnad = type=operator, script-path= RenameByUnad.js, argument=XXX
```

3. 修改脚本中的 `SUB_NAME` 变量为你想要的订阅名称

### 手动使用

直接调用 `operator()` 函数，传入代理列表：

```javascript
const renamedProxies = operator(originalProxies);
```

## 配置说明

### 修改订阅名称

编辑脚本第 2 行：

```javascript
const SUB_NAME = 'XXX'; // 修改为你想要的订阅名称
```

### 输出格式

- 有倍数标识：`订阅名 国家代码 - 编号 - 倍数x`
  - 示例：`XXX US - 01 - 10x`
  
- 无倍数标识：`订阅名 国家代码 - 编号`
  - 示例：`XXX JP - 02`

## 注意事项

1. 请确保脚本在代理处理流程中正确调用
2. 节点名称中的数字将被提取为编号（如 "节点123" → 编号 "123"）
3. 节点名称中的倍数标识（如 "10倍"、"10x"）将被提取并显示
